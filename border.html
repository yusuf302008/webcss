Font Style

CSS dapat mengatur style dari teks yang ada pada halaman web yang dibuat. Terdapat beberapa property yang dapat digunakan untuk mengatur style teks yaitu sebagi berikut.

  • text-align = digunakan untuk mengatur perataan teks, value yang dapat digunakan yaitu left, center, right
  • color = digunakan memberikan warna pada teks
  • letter-spacing = mengatur jarak antar karakter
  • word-spacing = mengatur jarak antar kata
